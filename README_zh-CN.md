# f-string

一个为 Rust 提供类似 Python f-string 字符串插值功能的过程宏库。

## 宏

| 宏    | 语法                | 说明         | 状态  |
|------|-------------------|------------|-----|
| `f!` | `f!("字符串 {表达式}")` | 带引号的字符串字面量 | 不稳定 |
| `t!` | `t!(字符串 {表达式})`   | 原生 Token 流 | 稳定  |

两种宏均在编译时展开为 `format!` 调用，无运行时开销。

## 示例

    // f! — 带引号，支持花括号转义
    let s = f!("Hello, {name}!");
    
    // t! — 无需引号，天然支持多行和双引号
    let s = t!(Hello, {name}!);

支持任意 Rust 表达式和所有 `format!` 格式化说明符。

## 限制

* 仅支持字符串字面量
* 需要 Rust 2021 edition
* `f!` 中使用 `{{}}` 转义花括号；`t!` 无法转义花括号
